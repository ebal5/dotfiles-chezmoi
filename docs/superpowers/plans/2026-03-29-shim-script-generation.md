# Shim Script Auto-Generation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** `chezmoi apply` 時に定義ファイルから uvx/bunx ラッパースクリプトを `~/.scripts/` に自動生成し、非対話環境でもツールを呼べるようにする

**Architecture:** 定義ファイル (`dot_config/shim-definitions`) をパースする `run_after_` シェルスクリプトが、マーカー付きラッパーを生成・クリーンアップする。既存の uvx/bunx エイリアスはラッパーに置き換えて削除する。

**Tech Stack:** POSIX sh, chezmoi templates (Go template)

### Task 2: Create the run_after_ shim generator script

**Files:**

- Create: `run_after_generate_shims.sh.tmpl`

- [ ] **Step 1: Create the generator script**

```bash
{{ if ne .chezmoi.os "windows" }}
#!/bin/sh
# Generate shim scripts from ~/.config/shim-definitions
# Each shim is a thin wrapper: exec <runner> <package> "$@"
# Managed shims contain marker "AUTO-GENERATED by chezmoi shim"

set -eu

DEFS="$HOME/.config/shim-definitions"
SHIM_DIR="$HOME/.scripts"
MARKER="# AUTO-GENERATED by chezmoi shim — do not edit"

# Exit silently if no definition file
[ -f "$DEFS" ] || exit 0

# Ensure shim directory exists
mkdir -p "$SHIM_DIR"

# Collect names of shims that should exist
valid_shims=""

while IFS= read -r line || [ -n "$line" ]; do
  # Skip comments and empty lines
  case "$line" in
    "#"* | "") continue ;;
  esac

  # Parse fields (split on :)
  runner="${line%%:*}"
  rest="${line#*:}"

  # Check for alias (third field)
  case "$rest" in
    *:*)
      package="${rest%%:*}"
      cmd="${rest#*:}"
      ;;
    *)
      package="$rest"
      cmd="$rest"
      # Reject @version without alias
      case "$package" in
        *@*)
          echo "shim-gen: ERROR: package '$package' contains @ but no alias specified, skipping" >&2
          continue
          ;;
      esac
      ;;
  esac

  # Validate
  if [ -z "$runner" ] || [ -z "$package" ] || [ -z "$cmd" ]; then
    echo "shim-gen: WARNING: invalid line '$line', skipping" >&2
    continue
  fi

  valid_shims="$valid_shims $cmd"

  # Generate shim content
  content="#!/bin/sh
$MARKER
exec $runner $package \"\$@\""

  target="$SHIM_DIR/$cmd"

  # Write only if different
  if [ -f "$target" ] && [ "$(cat "$target")" = "$content" ]; then
    continue
  fi

  printf '%s\n' "$content" > "$target"
  chmod +x "$target"
  echo "shim-gen: generated $target"
done < "$DEFS"

# Cleanup: remove stale auto-generated shims
for f in "$SHIM_DIR"/*; do
  [ -f "$f" ] || continue
  # Check for marker
  head -2 "$f" | grep -q "AUTO-GENERATED by chezmoi shim" || continue
  # Check if this shim is still in valid list
  name="$(basename "$f")"
  case " $valid_shims " in
    *" $name "*)
      # Still valid, keep
      ;;
    *)
      rm "$f"
      echo "shim-gen: removed stale shim $f"
      ;;
  esac
done
{{ end }}
```

Write this content to `run_after_generate_shims.sh.tmpl`.
